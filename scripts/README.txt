Neste arquivo comprimido adxúntanse os seguintes arquivos:

	- Informe da parte de deseño do sistema (base de datos, interfaz gráfica, etc.). 

Nel téñense todas as partes pedidas orixinalmente no guión da práctica: descrición do traballo, as transaccións, deseño da base de datos e a interface gráfica.
Adicionalmente, engadimos un pequeno apartado final no que facemos algunhas consideracións comúns que non sabíamos exactamente onde colocar, pero que cremos 
que resulta interesante e importante que se teñan en conta. Tratamos nel algunhas ideas sobre os scripts, un comentario sobre o deseño e algo sobre seguridade
que non chegamos a implementar pero que nos pareceu preciso mencionar.

	- Script de xeración da base de datos: nel téñense triggers e vistas tamén. En todo caso, despois da creación das táboas, 
introducíronse comentarios antes de cada trigger ou procedemento que se foi elaborando.

	- Script de borrado da base de datos: permitiranos eliminar todas as táboas creadas co script anterior.

	- Script de xeración de datos de proba: neste script, consideramos que temos os datos suficientes como para poder realizar
diferentes probas sobre AS DÚAS aplicacións. Deberase executar despois de crear as táboas.

Como se pode comprobar, polo tanto, decidimos empregar os dous grupos de implementación os mesmos scripts. Polo demais, non temos que
engadir nada mais. Todo o que haxa que considerar sobre o deseño da nosa base de datos está recollido no informe correspondente.
